# Interactive Music Player UI
# website link=
A visually appealing **music player UI** built with **HTML and CSS**, featuring interactive elements like hover animations, animated buttons, and a progress bar.  

This project is **UI-focused**, but it can be extended with JavaScript to add full music playback functionality.

---

## Features

- Smooth hover effects on the music player card and album cover  
- Interactive control buttons (Play, Pause, Next, Previous) with scaling animation  
- Animated progress bar to indicate song progress (static for now)  
- Modern glassmorphic design with blur and transparency  
- Fully responsive layout for desktop and mobile screens
